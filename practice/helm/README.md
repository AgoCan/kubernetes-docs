## What is Helm?

`helm` 帮助您管理Kubernetes应用程序  `Helm Charts`可以帮助您定义，安装和升级最复杂的Kubernetes应用程序。

charts 易于创建，版本控制，共享和发布-因此开始使用Helm并停止复制和粘贴。

CNCF将与Microsoft，Google，Bitnami和Helm贡献者社区合作维护最新版本的Helm。

### 以下是类比关系

|操作系统|包管理工具|包名|
|---|---|---|
|Ubuntu|apt|deb|
|CentOS|yum|rpm|
|Kubernetes|helm|charts|
